# Support

- **Questions / help**: open a [GitHub Discussion](https://github.com/namphuongtran/nami/discussions) or an issue using the question template.
- **Bugs**: open an issue with the bug template, including version/commit and reproduction steps.
- **Security vulnerabilities**: follow [SECURITY.md](SECURITY.md). Never open a public issue.

Nami is pre-1.0 community software with no commercial support offering. Response times are best-effort.
